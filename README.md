# 🧠 Human Mind Guide | دليل عقل الإنسان

A modern bilingual educational website about the human mind, subconscious programming, habits, self-development, and mental growth.

## ✨ Overview

This project is an interactive and visually modern website designed to explain:

* How the human mind works
* Conscious vs subconscious mind
* Mental programming
* Habit formation
* Influence of thoughts and environment
* Achieving goals using mindset principles

The website supports both:

* 🇸🇦 Arabic (RTL)
* 🇬🇧 English (LTR)

with a dynamic language switch system.

---

# 🚀 Features

* 🌍 Full bilingual support (Arabic / English)
* 🔄 RTL & LTR layout switching
* 📱 Fully responsive design
* 🎨 Modern futuristic UI
* ✨ Smooth animations and transitions
* 🧠 Educational content structure
* 📌 Interactive navigation
* 💾 Language preference saved using LocalStorage
* 📖 Single-page smooth scrolling experience
